## 📊 Architecture Overview

```
User Query
    ↓
Intent Classifier (What kind of query is this?)
    ↓
Provider Manager (Which providers are healthy?)
    ↓
Multi-LLM Router (Which model should I use?)
    ├─ Try Optimal Provider
    ├─ If fails → Try Alternative 1
    ├─ If fails → Try Alternative 2
    ├─ If fails → Try OpenRouter (final fallback)
    └─ If all fail → Return graceful error
    ↓
Cost Tracker (Record usage & costs)
    ↓
Response to User
```

## 💰 Cost-Aware Routing

**Provider Pricing (per 1M tokens):**
- Gemini Flash Lite: **$0.075** (fastest + cheapest)
- Gemini Flash: **$0.1**
- Gemini Reasoning: **$0.6** (only for complex tasks)
- Groq: **$0.59** (cost-effective)
- OpenRouter: Variable (fallback)

**Result:** Automatic cost optimization! Simple queries route to cheapest providers, complex queries only use premium models when needed.

## 🔧 Quick Integration

### 1. Install dependency
```bash
pip install google-generativeai
```

### 2. Set API keys (.env)
```
GEMINI_API_KEY=your_key
GROQ_API_KEY=your_key (optional)
OPENROUTER_API_KEY=your_key (optional)
```

### 3. Use in code
```python
from llm_router_gemini import get_multi_llm_router

router = get_multi_llm_router()
result = router.route_and_call(
    query="Hello!",
    messages=chat_history,
    conversation_length=len(chat_history)
)

if result["success"]:
    print(result["response"])
    print(f"Cost: ${result['cost']:.6f}")
```
