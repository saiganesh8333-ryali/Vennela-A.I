📖 QUICK START
================================================================================

1. VALIDATE SETUP
   python validate_setup.py

2. CONFIGURE
   cp .env.example .env
   # Edit .env with API keys:
   # - GROQ_API_KEY from https://console.groq.com
   # - OPENROUTER_API_KEY from https://openrouter.ai

3. RUN SERVER
   uvicorn main:app --reload

4. TEST API
   curl http://localhost:8000/health
   curl -X POST http://localhost:8000/chat \
     -H "Content-Type: application/json" \
     -d '{"user_id":"user1","message":"Hello!"}'

================================================================================
🎯 API ENDPOINTS
================================================================================

GET /health
  Returns: {status, firebase_status, version}

POST /chat
  Input: {user_id, message}
  Output: {reply, provider, relevant_memory, memory_summary, latency_ms}

GET /memory/{user_id}
  Returns: {profile, summary, emotions, sentiments, counts}
