🚀 HOW IT WORKS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

USER SENDS MESSAGE
    ↓
RATE LIMIT CHECK
    ↓
LOAD USER MEMORY
    ↓
SAVE USER MESSAGE
    ↓
🚀 TRY GROQ (Primary - Fast)
    ├─ ✅ SUCCESS
    │   └─ Return response (provider: "Groq")
    │       Latency: ~450ms
    │
    └─ ❌ FAILED
        ↓
    ⚠️ AUTOMATICALLY SWITCH TO OPENROUTER (Fallback - Reliable)
        ├─ ✅ SUCCESS
        │   └─ Return response (provider: "OpenRouter")
        │       Latency: ~820ms
        │
        └─ ❌ FAILED
            └─ Return friendly error message

RESULT: Zero user disruption. System is resilient.
