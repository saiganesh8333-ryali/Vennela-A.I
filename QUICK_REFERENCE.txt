## 🔑 Key Formulas

### Memory Importance (Phase 2)
```python
importance = (
    emotional_weight * 0.4 +
    repetition_weight * 0.3 +
    recency_weight * 0.3
)
# Result: 0.0 (unimportant) to 1.0 (critical)
```

### Multi-Signal Reward (Phase 3)
```python
reward = (
    0.3 * user_engaged +
    0.3 * response_helpful +
    0.2 * prediction_accurate +
    0.1 * response_time_good +
    0.1 * suggestion_accepted
)
# Result: -1.0 (bad) to +1.0 (good)
```

## ⚠️ Common Issues & Fixes

| Issue | Cause | Fix |
|-------|-------|-----|
| ImportError | Missing file | Check file in root directory |
| Circular import | Bad dependency | Already fixed in Phase 1 |
| Memory high | Too much history | Reduce max_history parameter |
| Predictions wrong | Not enough data | Wait 10-20 interactions |
| No patterns | Just started | Wait 5+ interactions |

## 🎓 Architecture Diagram

```
User Query
    ↓
Phase 1: LLM Router (pick model)
    ↓
Phase 2: Pattern Detector (learn profile)
    ↓
Phase 3: Prediction Engine (predict next move)
    ↓
Phase 3: RL Tracker (record feedback)
    ↓
Adapt Response (based on phases 1-3)
    ↓
Return Response
    ↓
Record Learning Data (Firebase)
```
