💡 HOW IMPORT REDIRECTION WORKS
═════════════════════════════════════════════════════════════════════════

When app.py starts:
  1. lightweight_redirect.py is imported
  2. It patches Python's import system
  3. It intercepts heavy library imports
  4. It redirects to lightweight versions

Example:
  Code: from sentence_transformers import SentenceTransformer
  Intercept: Wait, need SentenceTransformer?
  Redirect: → lightweight_embeddings.SentenceTransformer
  Result: Works perfectly, code unchanged ✓
